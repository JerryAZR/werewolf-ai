# 狼人杀游戏规则

## 游戏概述

狼人杀是一款多人社交推理游戏，分为狼人阵营和村民阵营。狼人隐藏在村民中，每到夜晚狼人杀死一名村民，白天所有玩家通过发言投票找出狼人。

村民阵营需要找出会伪装自己的狼人并将其放逐；狼人阵营则需要通过欺骗和协作消灭村民。

## 屠边规则

狼人阵营有两种胜利方式：
- **屠城**：杀光所有村民和神职
- **屠边**：杀光所有神职，或杀光所有普通村民

游戏采用**屠边规则**，即当任意一方（普通村民或神职）全部出局时，狼人立即获胜。村民阵营需要消灭所有狼人才能获胜。

## 角色配置（12人标准版）

### 狼人阵营（4人）
| 角色 | 能力描述 |
|------|----------|
| 狼人 | 每晚可共同杀死一名玩家 |

### 村民阵营（8人）
| 角色 | 数量 | 能力描述 |
|------|------|----------|
| 普通村民 | 4 | 无特殊能力，白天参与投票和讨论 |
| 预言家 | 1 | 每晚可查验一名玩家的身份（好人/狼人） |
| 女巫 | 1 | 拥有一瓶解药和一瓶毒药，每晚可救一人或毒一人 |
| 猎人 | 1 | 被狼人杀死后可开枪带走一名玩家（被女巫毒杀不能发动） |
| 守卫 | 1 | 每晚可守护一名玩家，使其免受狼人伤害（不能连续守护同一人） |

### 警长（称号）
警长**不是独立角色**，而是可授予任何玩家的**称号**。第一日可选举一名玩家为警长，该玩家获得以下能力：
- 投票计1.5票（可打破平局）
- 主持讨论
- 警徽可传递

## 游戏流程

游戏分为白天和夜晚两个阶段，以"天"为单位循环进行。游戏从**第一晚**开始。

每轮循环的结构为：**夜晚阶段 → 白天阶段 → 夜晚阶段 → 白天阶段 → ...**

无论第一晚还是后续夜晚，每个夜晚结束后必定进入白天阶段进行讨论和投票。

### 夜晚阶段

由于采用电子游戏实现，各角色行动无需严格按顺序执行，夜间行动可并行处理，只需注意以下约束：

- **女巫**需要在狼人做出击杀决策后才能行动，以便决定是否使用解药
- **守卫**和**预言家**的行动时机无特殊限制，可在任意时刻执行
- 夜间除女巫可获知狼人击杀目标外，其他角色的行动结果均保密

#### 夜间规则说明：
- **狼人**首先决定击杀目标
- **女巫**随后行动，知晓狼人击杀目标后决定是否使用解药或毒药
- **守卫的守护对女巫毒药无效**，被毒玩家必定死亡
- 若守卫守护的玩家同时被女巫用解药救活，该玩家必定存活
- **若女巫毒杀守卫，守卫当夜的守护技能仍然有效**
- 女巫与守卫在夜晚行动时互相不知道对方的行动
- 狼人可以看到同伴，夜间作为一个整体决定击杀目标
- 女巫可以看到被狼人杀死的玩家
### 白天阶段

1. **警长竞选（仅第一日）**：第一日**首先**进行警长选举，**此时尚未公布夜晚死亡情况**，因此**任何玩家均可参选**（包括夜晚已死亡但尚未被宣布的玩家）
2. **公布昨夜情况**：宣布死亡玩家，但不公开死因和身份
3. **遗言环节**：昨晚死亡玩家发表遗言（每位玩家整局游戏只有一次遗言机会）
4. **发言讨论**：存活玩家按顺序发言，警长负责组织讨论
5. **投票放逐**：所有玩家投票放逐一名嫌疑人
6. **放逐结算**：票数最多者被放逐
7. **胜负判定**：检查是否满足胜利条件

## 死亡机制

- **狼人击杀**：被狼人选中的玩家夜晚死亡
- **女巫毒杀**：被女巫用毒药杀死的玩家夜晚死亡
- **放逐投票**：白天投票最多的玩家被放逐
- **猎人枪杀**：猎人被狼人击杀后可选择开枪带走一名玩家

## 遗言规则

- 夜晚死亡的角色：**第一晚**死亡有遗言机会，**第二晚及之后**死亡没有遗言
- 白天被投票放逐的角色：**始终有遗言机会**
- **夜晚多死**：若多玩家夜晚死亡，按座位顺序发表遗言
- 遗言在死亡后立即发表，内容可包含对其他玩家的指认和分析
- 发表遗言后该玩家不再参与后续游戏

## 投票规则

- 投票阶段所有存活玩家必须投票，**允许弃票**
- 平票处理：票数最多的玩家进入平票PK环节
- PK规则：平票玩家重新发言，随后其他玩家再次投票
- 若再次投票仍无结果，则本轮不进行放逐

## 暗牌规则

玩家出局时**不公开身份牌**。其他玩家只能通过该玩家的发言和行为来推断其身份。

因此，分析玩家遗言中透露的信息成为判断局势的重要依据。

## 胜利条件

### 村民阵营胜利
- 所有狼人被放逐或死亡

### 狼人阵营胜利
- 所有普通村民死亡，或所有神职死亡（屠边规则）

## 游戏流程概览

```
游戏开始
    │
    ▼
第一晚 → 守卫行动 → 预言家查验 → 女巫用药 → 狼人击杀
    │
    ▼
第一日白天 → 警长竞选 → 公布死讯 → 遗言 → 发言讨论 → 投票放逐
    │
    ▼
胜负判定 ─┬─ 村民胜利 → 游戏结束
         │
         ▼
    第二日及之后 → 公布死讯 → 遗言 → 发言讨论 → 投票放逐
         │
         ▼
    胜负判定 ─┬─ 村民胜利 → 游戏结束
              │
              ▼
         夜晚阶段 → 循环...
              │
              ▼
         狼人胜利 → 游戏结束
```

## 角色详细说明

### 预言家
- 每晚可选择一名存活玩家查验其身份
- 查验结果为"好人"或"狼人"
- 建议在白天尽早亮明身份并报查验信息
- 需警惕狼人假扮预言家混淆视听

### 女巫
- 拥有一瓶解药（救活被狼人杀死的玩家）和一瓶毒药（毒杀任意玩家）
- 每晚**只能使用一瓶药**，即救人和毒药不能同一晚使用
- 第一晚通常建议救人（若狼人击杀目标威胁较大）
- 解药对自己无效
- 毒药可以毒杀狼人、神职或村民
- **毒药无视守卫守护**，被毒玩家必定死亡
- **毒杀守卫不影响守卫当夜已发动的守护技能**

### 猎人
- 被狼人杀死后可立即开枪带走一名玩家
- 被放逐投票处决时可选择开枪带走一名玩家
- 被女巫毒杀时不能发动技能
- **可以主动放弃使用**
- **目标规则**：可选择任意存活玩家为目标，只能通过玩家名称/ID指定，不能通过角色或阵营指定
- 建议在确认狼人身份后谨慎使用技能

### 守卫
- 每晚可守护一名玩家，使其免受狼人伤害
- **不能连续两晚守护同一人**
- 若守护和女巫救的是同一人，该玩家必定存活
- **守护对女巫毒药无效**，被毒玩家必定死亡
- 即使被女巫毒杀，当夜守护技能仍生效
- 守卫身份暴露后可能成为狼人重点击杀/毒杀目标

### 警长
- **竞选时间**：第一日白天最先进行警长选举，投票结束后再公布夜晚情况
- **投票权重**：警长投票始终计 **1.5票**（而非1票），可有效打破平局
- **竞选资格**：第一日白天警长竞选在死亡宣布**之前**进行，**任何玩家均可参选**（包括夜晚已死亡但尚未被宣布的玩家）
- **主持讨论**：警长负责组织和管理白天的讨论流程
- **PK环节**：PK环节中，警长投票仍计1.5票
- **警徽传递**：警长被放逐或夜晚死亡时，可选择将警徽传给场上存活的任意一名玩家
- **不影响阵营归属**：警长身份**不改变**玩家原有阵营。狼人当选警长仍与狼人阵营共同胜利
- **死亡规则**：警长被放逐时有遗言，遗言后可指定警徽继承人
- **夜袭死亡**：若警长夜晚被杀，警徽信息不公开，**次日早上**公布夜晚结果时可指定警徽继承人

### 狼人协同（多狼规则）
- 夜间所有存活狼人视为一个整体，共同决定击杀目标
- 狼人之间互相知道身份，协商后统一行动
- 若狼人中有 AI 控制也有玩家控制，AI 决策仅提供参考，最终决定权在玩家
- **本版本所有玩家均为 AI**，狼人集体通过一次 AI 调用完成决策

## 争议规则说明

狼人杀在不同玩家群体中可能存在规则分歧，本游戏采用以下规则：

### 游戏基础规则
| 规则项 | 我们实现 |
|--------|----------|
| 胜负规则 | 屠边规则（狼人杀光神职或杀光普通村民即获胜） |
| 身份规则 | 暗牌规则（玩家出局时不公开身份牌） |
| 游戏起始 | 从第一晚开始，**第一日白天进行警长竞选** |
| 白天流程 | 每轮循环均包含完整白天阶段（讨论+投票），无区别 |

### 夜间行动规则
| 规则项 | 我们实现 |
|--------|----------|
| 行动顺序 | 狼人先行动，女巫后行动；守卫和预言家可与狼人并行处理 |
| 信息共享 | 女巫与守卫互相不知道对方的夜间行动；女巫可获知狼人击杀目标 |
| 狼人空刀 | 允许（狼人可选择不击杀任何玩家） |
| 平安夜 | 只要夜间结算后无玩家死亡，即宣布"昨晚是平安夜" |
| 狼人协同 | 所有狼人作为整体行动，统一决定击杀目标 |
| 守卫与毒药 | 守卫不能保护玩家免受女巫毒药；守卫被毒杀后技能仍生效 |

### 角色技能规则
| 规则项 | 我们实现 |
|--------|----------|
| 警长竞选 | 第一日死亡宣布**之前**进行；任何玩家均可参选（包括夜晚已死亡但尚未被宣布的玩家）；投票不允许弃票；多数票当选 |
| 警长权力 | 归票权（平票时打破平局，PK环节仍计1.5票）；主持讨论；被放逐时可指定警徽继承人 |
| 猎人技能 | 被狼人击杀或被放逐时可发动；被毒杀时不能发动；**可以主动放弃使用** |
| 守卫守护 | 不能连续两晚守护同一人；**允许自守** |
| 女巫用药 | 每晚只能使用一瓶药（救人和毒药不能同一晚使用）；解药对自己无效；**第一晚允许用毒药** |

### 投票与遗言规则
| 规则项 | 我们实现 |
|--------|----------|
| 投票方式 | 允许弃票 |
| 平票处理 | 进入平票PK，重新发言后投票；仍平票则本轮不放逐 |
| 夜晚遗言 | 仅第一晚死亡的玩家有遗言 |
| 白天遗言 | 被放逐的玩家始终有遗言 |

### 未实现规则
以下为常见规则，但本项目暂不实现：
| 规则项 | 说明 |
|--------|------|
| 狼人自爆 | 狼人不能在白天自爆直接进入夜晚 |
